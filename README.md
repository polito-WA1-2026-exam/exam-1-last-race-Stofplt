# Last Race

Project for Web Applications I 2025/26.

This repository follows the required structure:

- `client/`: React 19 single-page application.
- `server/`: Node.js + Express HTTP API with SQLite.

## Development

Start the server:

```bash
cd server
npm run init-db
nodemon index.js
```

Start the client:

```bash
cd client
npm run dev
```
