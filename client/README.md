# Last Race Client

React application for Last Race, built with Vite.

## Scripts

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Notes

- The client expects the API server to run on `http://localhost:3001`.
- The development server runs on `http://localhost:5173` by default.
- Main application routes and project-level documentation are described in the root `README.md`.
